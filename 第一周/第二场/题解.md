
# 题解
## 第一题(思维):
### 题意: 
输入一组木棍的坐标。
对任何一个木棍做的操作:
1.花费1金币让当前木棍的前进或者后退1格。
2.免费让一个木棍前进后退2格
花费最少金币让所有木棍位于同一坐标。

### 解法:

因为奇数+1 变成偶数,偶数+1变成奇数。
并且木棍前进后退2格是不用花费金币的。
所以我们只要统计奇数和偶数的个数，然后求两者的最小值。

```
#include <iostream>
#include <algorithm>
using namespace std;
int main() {
	int n;
	cin >> n;
	int cnt = 0;
	for (int i = 0; i < n; ++i) {
		int x;
		cin >> x;
		cnt += x & 1;
	}
	cout << min(cnt, n - cnt) << endl;
	return 0;
}
```
